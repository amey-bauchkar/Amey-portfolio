## 💻 Featured Projects

### 1. AquaLens (Intelligent Water Quality Monitor)
A premium, Apple-inspired interface with strict SSOT state-driven architecture. Connects to ESP8266 + Firebase Realtime Database with a smart decision engine for flow control and automatic escalation alerts.

### 2. Line Follower Robot
An autonomous robotic vehicle designed to navigate a predefined path. Built and coded from scratch for college competitions, featuring advanced PID tuning and glare protection logic.

### 3. Self Balancing Robot
A two-wheeled robot using an MPU6050 sensor and Cascaded PID control algorithm (Inner loop for Angle, Outer loop for Velocity) to autonomously maintain its upright balance.

### 4. Smart Dustbin Automation
An automated waste management solution utilizing ultrasonic sensors to detect proximity and open the lid automatically via servo actuation, complete with a non-blocking state machine.

### 5. Crane Safety System
A fail-safe hardware prototype designed to monitor weight loads (using an HX711 and Load Cell) and prevent tipping hazards in industrial crane operations with real-time telemetry and emergency cutoff.

## 🚀 Getting Started (Local Development)

To run this portfolio locally on your machine, follow these steps:

### Prerequisites
- [Node.js](https://nodejs.org/) installed (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amey-bauchkar/Amey-portfolio.git
   cd Amey-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
